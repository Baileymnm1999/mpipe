Installation instructionsÂ¶

The easiest way to install MPipe on your system is using pip command:

pip install mpipe

Or you may provide --user flag if you don’t have root privs:

pip install --user mpipe

Another way is using easy_install:

easy_install mpipe

Or, you can clone the GitHub repo and manually run the Distutils installer:

git clone http://github.com/vmlaker/mpipe
cd mpipe
python setup.py install --user

Another way to get the source code is to download the tarball from the Package Index at http://pypi.python.org/pypi/mpipe.
