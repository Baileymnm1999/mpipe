About MPipe¶

MPipe (pronounced “em pipe”) is written by Velimir Mlaker. It’s a small, lightweight Python library you can use to create programs organized as multitasking pipelined workflows. MPipe is written in 100% pure Python.

The web pages you are reading were created with the fabulous Sphinx documentation generator. Pictures in the documentation were drawn with Dia, the powerful and easy-to-use diagram creation program.
