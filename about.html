About MPipeÂ¶

MPipe (pronounced “em pipe”) is written by Velimir Mlaker. It’s a tiny Python library you can use to create programs organized as multitasking pipelined workflows. MPipe is written purely in Python.

The web pages you are reading were created with the fabulous Sphinx documentation generator. Pictures in the documentation were drawn with Dia, the powerful and easy-to-use diagram creation program.

MPipe is an open source GitHub project at github.com/vmlaker/mpipe.
